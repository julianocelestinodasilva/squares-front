This is a Front End that list the territories exposed by Squares API.
This app was developed usging Node JS and Angular JS (v1.3.15).

	- To Run the application :
  		To run the app is necessary to have Node (npm) installed on machine. At the first time, is necessary to run the command "npm-install" to install the necessary packages to run the app. After executed the "npm-install" (only at the first time), execute "node server.js" (both commands are executed in the root folder).

	- Oen App in the browser :
  		http://localhost:3000/
